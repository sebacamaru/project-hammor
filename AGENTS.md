# Repository Instructions

Antes de empezar cualquier tarea en este repositorio, leé estos archivos para tomar contexto del proyecto:
- `CLAUDE.md`
- `docs/architecture.md`
- `docs/memory.md`

Reglas:
- Tratá `CLAUDE.md` como contexto principal del proyecto.
- Usá `docs/architecture.md` para detalles de arquitectura y subsistemas.
- Usá `docs/memory.md` como índice de archivos clave, estado actual y continuidad entre sesiones.
- Si una tarea afecta arquitectura, flujo de runtime, editor, networking o persistencia, revisá nuevamente los archivos relevantes antes de editar.
